===================
Tutorials
===================

Step-by-step tutorials for common PaperSorter integrations and advanced use cases. These hands-on guides walk you through real-world scenarios with detailed instructions and examples.

Each tutorial is designed to be self-contained and includes all necessary configuration files, code snippets, and troubleshooting tips.

What You'll Find Here
=====================

Integration Guides
------------------

Learn how to connect PaperSorter with popular services and tools:

- **Email Newsletter Setup**: Configure Gmail or other email providers
- **Slack Integration**: Set up team notifications and channels
- **Custom Embedding Models**: Use local or specialized embedding APIs
- **Multi-Model Workflows**: Manage different models for different research areas

These tutorials assume you have completed the :doc:`../getting-started/index` guide and have a working PaperSorter installation.

Tutorial Structure
==================

Each tutorial follows a consistent format:

**Prerequisites**
  What you need before starting (accounts, API keys, etc.)

**Overview**
  What you'll accomplish and why it's useful

**Step-by-Step Instructions**
  Detailed walkthrough with commands and configuration

**Testing and Verification**
  How to confirm everything is working correctly

**Troubleshooting**
  Common issues and their solutions

**Next Steps**
  Related tutorials and advanced configurations

Prerequisites
=============

Before starting any tutorial:

- Complete the :doc:`../getting-started/quickstart` guide
- Have a working PaperSorter installation
- Access to necessary external services (Gmail, Slack, etc.)
- Basic familiarity with configuration files and command-line tools

Getting Help
============

If you encounter issues:

1. Check the tutorial's troubleshooting section
2. Review the :doc:`../admin-guide/troubleshooting` guide
3. Search existing GitHub issues
4. Ask for help in community discussions
